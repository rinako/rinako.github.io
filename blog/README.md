  <strong>一、github使用方法</strong><br/>
  <strong>① 编辑、下载、删除文件</strong><br/>
  点击目录中的文件或文件夹，会进入资源管理器界面，然后就可以在右边编辑或下载。<br/>
  更改文件名及类型，请在路径里修改。<br/>
  想好位置再添加文件，移动起来很麻烦。<br/>
  点击【...】打开菜单，选择删除。<br/>
  <strong>注意</strong>：所有操作，<strong>最后都要点绿色的按钮进行再次确认！不然不会保存或修改！</strong>
</p><p>
  <strong>② 添加文件</strong><br/>
  github中不能直接创建文件夹，<strong>只能创建文件夹下的文件</strong>。<br/>
  注意文件编辑界面上的<strong>路径</strong>：<br/>
  在空格处写<strong>【文件名+后缀名】</strong>能直接创建文件。
</p><p>
  <strong>③ 添加文件夹</strong><br/>
  可以利用路径添加文件夹。<br/>
  例：在root文件夹下添加名为file文件夹，需要先文件夹下创建0.txt的空文件。<br>
  路径：root/file/0.txt<br/>
  当然也能直接创建有用的文件，节省操作步骤。<br/>
  <br/>
  但<strong>不能一次性添加下级目录</strong>，如：<br/>
  root/file/mp3/0.txt<br/>
  这样会创建出一个<strong>【file/mp3】</strong>的文件夹。<br/>
</p><p>
  <strong>④ 删除文件夹</strong><br/>
  下拉<stong>【...】菜单</stong>，选中<strong>【Delete directory】</strong>。<br/>
</p><p>
  <strong>⑤ 上传文件</strong><br/>
  重点是在新文件夹中上传，一定要<strong>先创建好文件夹 → ③</strong>。
</p><p>
